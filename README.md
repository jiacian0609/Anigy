## localhost 安裝與測試步驟
### 環境設置
#### 設定環境變數
1. 下載[環境變數壓縮檔](https://drive.google.com/file/d/1Kc9bn1kCnH_t9AmGpybCQsSSMzHE2M17/view?usp=sharing)
2. 將兩個 `.env` 檔分別放到 `/frontend` 以及 `/backend` 底下
#### 啟動後端
1. 安裝所需套件
	```
	cd backend
	npm install
	```
	或
	```
	cd backend
	yarn
	```
2. 啟動伺服器
	```
	yarn server
	```
	或
	```
	npm run server
	```
#### 啟動前端
1. 安裝所需套件
	```
	cd frontend
	npm install
	```
	或
	```
	cd frontend
	yarn
	```
2. 啟動伺服器
	```
	yarn start
	```
	或
	```
	npm start
	```
