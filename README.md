## localhost 安裝與測試步驟
### 環境設置
1. 下載 `.env` 檔
2. 安裝所需套件  
   分別 `cd backend` 以及 `cd frontend` 並執行 `yarn` 以安裝所需套件
3. 啟動前後端  
   於本專案的根目錄開啟兩個終端機，分別執行 `yarn start` 以及 `yarn server` 以開啟前後端
